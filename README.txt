This is my flask Web Application, it is a simple Stock Tracker website that uses the Yahoo finance API to 
deliver Detailed Stock info for the user

To run the project the following commands are required. 

==========================================================================================================

1.
pip install requests

2.
pip install Flask Flask-WTF WTForms[email] requests python-dotenv

3.
pip install yfinance

4.
python app.py

==========================================================================================================

Project will run on:
http://127.0.0.1:5000
http://192.168.4.85:5000
